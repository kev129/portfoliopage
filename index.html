Flight Information Checker

This JavaScript code powers a user-friendly Flight Information Checker web application. It dynamically fetches and updates real-time flight details, providing users with instant access to departure/arrival times and flight statuses. The project aims to streamline the flight-checking process, offering departure information, delays and weather at the destination so the user can plan their clothing appropriately.

The frontend is hosted using Github pages.

The backend is hosted using Render.

Deepfake Detector

The aim of this project is to recreate the model implemented in my final year dissertation and make it a fullstack application, with a frontend site where users can upload an image, and the image be processed in the backend, finally returning a percentage probability of it being a deepfake.

Currently the project is still in progress, with the backend model completed and able to return a probability, the next deliverable shall be to develop a frontend and link the two for a fully functioning site.
